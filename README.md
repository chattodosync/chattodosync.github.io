# chattodosync.github.io

Marketing website for Chattodo.

## Current messaging

The site reflects Chattodo's latest product direction:

- Focus + Later inbox model
- Chat, Todo, Team as the core pillars
- The Pulse as lightweight focus insight
- anti-bloat, clarity-first product philosophy
